# Favicon Instructions

To add a favicon to your website:

1. **Create or download a favicon**
   - Size: 16x16 or 32x32 pixels
   - Format: .ico, .png, or .svg
   - Tool: Use https://favicon.io/ to generate

2. **Save the file**
   - Place it in the root directory as `favicon.ico`
   - Or use PNG: `favicon.png`

3. **Add to HTML** (already included in all pages)
   ```html
   <link rel="icon" type="image/x-icon" href="favicon.ico">
   ```

4. **Alternative formats**
   ```html
   <link rel="icon" type="image/png" href="favicon.png">
   <link rel="icon" type="image/svg+xml" href="favicon.svg">
   ```
